HealthScribe PRO is an AI-powered medical documentation assistant that listens to doctor-patient conversations and automatically generates structured notes, prescriptions, and lab orders in real-time. ### Prerequisites
Make sure you have Node.js installed.

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Amxnnn/HealthScribe.git
   cd HealthScribe
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your API key:
   ```
   VITE_TAMBO_API_KEY=your_api_key_here
   ```

4. Run the project:
   ```bash
   npm run dev
   ```

That's it! Open the local link (usually `http://localhost:5173`) to start using the app.
